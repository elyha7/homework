Требования к проекту Project_avito:

Постановка задачи: имеются данные с сайта avito, которые представляют собой объявления в формате: заголовок, описание, числовые данные(даты, цены и тд). Часть объявлений запрещены для публикации на сайте avito законами РФ. Требуется провести классификацию объявлений на разрешенные для публикации и запрещеные.

Уточнение задачи:
1)Для выполнения задачи будут использоваться методы машинного обучения, такие как сверточные и рекурентные нейронные сети.
2)Задача выполняется на языке python с использованием библиотек машинного обучения theano и lasagne.
3)В качестве метрики точности предсказаний будут использованы auc score и accuracy.
4)Предполагается провести небольшое созтязание сверточных и рекурентных нейронных сетей и выяснить какие из них лучше себя показывают в такого рода задачах.
5)Предполагается создание небольшого интерфейса пользователя, который будет давать возможность загрузить свой датасет объявлений и выделить из них разрешенные.
